## Activity 1 – Coding Challenge

### Platform Used
**HackerRank / LeetCode** (online competitive programming platform)

### Challenge Completed
- **Category:** Problem Solving / Algorithms
- **Language Used:** Python

### Sample Problem: Hello World Program

```python
# Simple Python program demonstrating basic coding skills
def greet(name):
    """Returns a greeting message."""
    return f"Hello, {name}! Welcome to Digital Literacy."

# Main execution
if __name__ == "__main__":
    print(greet("World"))
    print("Digital literacy empowers individuals to thrive in a digital society.")
```

### Skills Demonstrated
- Basic programming syntax
- Function definition and usage
- String formatting
- Commenting and documentation
